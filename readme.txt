== Description ==

Chrysoberyl is a modern, clean, and fully responsive WordPress theme designed for news websites and blogs. It features:

* Responsive design that works on all devices
* Custom menu support with multiple locations
* Featured images support
* Post formats support
* Translation ready (Thai/English)
* Custom logo support
* Custom background and header support
* Widget areas in sidebar and footer
* Social sharing functionality
* Table of contents for long articles
* Search functionality with live suggestions
* Clean typography with Google Fonts

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

== Frequently Asked Questions ==

= Does this theme support any plugins? =

Chrysoberyl includes support for WooCommerce, Jetpack, and other popular plugins.

= Where can I find theme options? =

Theme options can be found under Appearance > Customize in your WordPress admin panel.

= How do I add icons to the theme? =

The theme uses icon classes that work with Font Awesome. To display icons, you can install a Font Awesome plugin such as "Font Awesome" from the WordPress plugin repository. The theme will work without icons if no icon plugin is installed.

= How do I add code syntax highlighting? =

For code syntax highlighting in posts, you can install a plugin like "Prism Syntax Highlighter" or "Enlighter" from the WordPress plugin repository.
